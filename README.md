# CITY EMERGENCY ALERTS V6.0

**Pune District Emergency Intelligence**

CITY EMERGENCY ALERTS is a privacy-first public emergency intelligence web app for Pune City, PCMC, Pune District, and nearby highways/ghats.

## Current status

**Production Stabilization Update 1**  
Status: Production Beta Candidate

## Tabs

- Situation
- Alerts
- Incidents
- Journey
- Official
- Emergency

## Key capabilities

- Current situation snapshot
- Weather and environmental intelligence
- Alerts and incidents
- River and dam intelligence
- Journey suitability scoring
- Official and trusted source links
- Emergency dialing
- Nearby emergency services
- Share My Location
- GitHub Pages deployment
- GitHub Actions intelligence pipeline

## Source philosophy

- Tier 1: official sources and emergency services
- Tier 2: trusted news sources for early awareness
- Tier 3: supporting signals only

## Citizen promise

Simple interface, accurate information, useful guidance.
